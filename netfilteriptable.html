NetFilter/IPtable

Netfilter is a framework that is part of the Linux Kernel used in conjunction with IPtable form the basis of Linux's network filtering system and allows user programs to hook into the kernel. The Netfilter allows users to filter datagrams, define address translation and determine processing operations. [5,9]

A simple firewall using Netfilter/IPtable design proposed by Wang et.al also includes state detection technology first proposed by the CheckPoint Company, thus adding the functionality of stateful packet filters, and a logging system so system administers can detect errors. The implementation of a firewall created using Netfilter/IPtable consists of 5 steps:

  1. Configure the kernel with the proper setting to us Netfilter/IPtable.
  2. Create a shell script to configure and start Netfilter/IPtable.
  3. Delete any existing rules for the filter so that the new rules do not conflict with the default rules.
  4. Change the default policies to match your security policies.
  5. Reset and stop the firewall to refresh settings.[5,9]

Netfilter allows for the addition of new modules, so Wang et.al added a simple anti-DDoS module to have added protection without sacrificing too much performance of their system.
